1. What is MCP?
H
Host
C
Client
S
Server
T
Transport
MCP Host
Application that wants AI + tool access. Examples: Claude Desktop, VS Code, Cursor. Coordinates multiple MCP client connections.
MCP Client
Protocol layer with 1:1 connection to one server. Handles JSON-RPC communication. Built into the host application.
MCP Server
Lightweight program exposing tools, resources, or prompts. Runs as a separate process (or in-process). Speaks the MCP protocol.
Transport Layer
stdio — subprocess pipe for local servers.
HTTP/SSE — network transport for remote/deployed servers.
The Analogy

MCP is the "USB-C port for AI" — one universal standard that lets any AI model plug into any tool, database, or service. Like HTTP standardized the web, MCP standardizes how AI connects to the world.

2. The 4 MCP Capabilities
Tools — Model-controlled
Functions the AI decides to call. Like function calling — LLM triggers execution.
@mcp.tool()
Resources — Application-controlled
Data the host app exposes to context. Files, DB rows, API data. URI-addressed.
@mcp.resource("uri://")
Prompts — User-controlled
Reusable prompt templates users invoke. Slash commands in Claude Desktop.
@server.list_prompts()
Sampling — Server-initiated
Server requests LLM completions. Enables agentic behavior inside servers.
session.create_message()
3. MCP Lifecycle
1
Initialize
Client sends capabilities + version. Server responds with its capabilities.
2
Handshake
Client sends notifications/initialized. Both sides ready.
3
Operate
Tool calls, resource reads, prompt requests, sampling — all via JSON-RPC 2.0.
4
Shutdown
Client sends notifications/closed. Server cleans up.
4. Core Code Patterns
FastMCP Server Pattern
#!/usr/bin/env -S uv run --script
# /// script
# requires-python = ">=3.9"
# dependencies = ["mcp[cli]>=1.0.0"]
# ///

from mcp.server.fastmcp import FastMCP

mcp = FastMCP("server-name")

@mcp.tool()
def my_tool(param: str) -> str:
    """Description visible to LLM"""
    return result

@mcp.resource("data://{key}")
def get_resource(key: str) -> str:
    """Resource description"""
    return load_data(key)

if __name__ == "__main__":
    mcp.run(transport="stdio")
Claude Agent SDK Pattern
import asyncio
from claude_agent_sdk import (
    Agent, tool, create_sdk_mcp_server
)

# Custom tool with decorator
@tool
def my_tool(input: str) -> str:
    """Tool available to the agent"""
    return process(input)

# In-process MCP server (no subprocess)
server = create_sdk_mcp_server("analysis")

@server.tool()
def analyze(data: str) -> str:
    return run_analysis(data)

# Run the agent
agent = Agent(
    name="helper",
    instructions="System instructions",
    tools=[my_tool],
    model="claude-sonnet-4-5"
)
asyncio.run(agent.run("Task"))

Security: PreToolUse Hook
from claude_agent_sdk import PreToolUse

def block_dangerous(tool_use):
    """Block rm -rf and similar commands"""
    if tool_use.tool_name == "Bash":
        cmd = tool_use.input.get("command","")
        dangerous = ["rm -rf", "sudo rm",
                     "format", "> /dev/"]
        for d in dangerous:
            if d in cmd:
                return {"type": "block",
                        "reason": "Blocked"}
    return {"type": "allow"}

# Attach to agent
agent = Agent(
    hooks=[PreToolUse(block_dangerous)],
    ...
)

6. Security Checklist & Common Gotchas
Security Checklist
  • Validate file paths — prevent directory traversal (.., leading /)
  • Use PreToolUse hooks to block dangerous Bash commands
  • Principle of least privilege — grant only needed tools
  • Use HTTPS for remote server transport
  • Audit log all sensitive operations
  • Rate-limit tool calls in production
  • Never commit .env files (API keys)
Common Gotchas
  • All MCP ops are async — don't forget await
  • Use absolute paths — relative paths break when server runs from a different directory
  • Resource URIs must be valid: file:///path not /path
  • "Method not found" → server doesn't implement that capability
  • "Invalid params" → check your JSON schema / type hints
  • Vercel hobby plan: 10s timeout — upgrade to Pro for Agent SDK cold starts

Key Takeaways
  • MCP = USB-C for AI: One protocol connects any AI model to any tool, database, or service
  • 4 Capabilities: Tools (model runs), Resources (app exposes), Prompts (user invokes), Sampling (server asks LLM)
  • FastMCP: Use @mcp.tool() decorators — Python type hints auto-generate the JSON schema
  • In-process servers: create_sdk_mcp_server() runs MCP tools in the same Python process — no subprocess overhead
  • Security first: Always validate paths, use PreToolUse hooks, grant least privilege, log everything
  • Deploy anywhere: FastAPI + Claude Agent SDK + SSE streaming = production-ready agent in <200 lines
  • UV handles deps: Inline # /// script headers mean zero setup — uv run server.py just works
