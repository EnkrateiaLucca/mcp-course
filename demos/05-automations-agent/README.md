# Automation Generator Demo

Simple Claude Agent + MCP demo for generating automation scripts.

## Quick Start

```bash
# Set API key
export ANTHROPIC_API_KEY='your-key'

# Run the agent
uv run automation_agent.py
```

## Files

- `automation_agent.py` - Claude agent (main entry point)
- `automation_mcp_server.py` - MCP server with database tools
- `automations_database.csv` - Script templates database
